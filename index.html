    Built entirely with React, Quantro prioritizes simplicity, speed, and privacy by storing all data locally in the browser. No sign-up or backend required.

    Currently, the app uses localStorage to manage data. However, future plans include transitioning Quantro into a full-stack application with user accounts, authentication, and external database support—enabling seamless access across multiple devices.
